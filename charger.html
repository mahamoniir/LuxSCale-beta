System Configuration

Problem: Rapid Battery Drain

Your LED PCB is rated 35V / 3000mA = 105W. The DM160-U on a 24V system supports up to 160W—so the controller isn't limiting you. You're running the LED at full 3A draw. Reducing to 1500mA halves the wattage to ~52.5W, effectively doubling your runtime per charge cycle.

Configuration Steps

  1. Wake the CU-ALL Remote

    Press any key to activate the LCD. Point directly at the controller's IR receiver window. Maintain distance under 3 meters for reliable signal transmission.

  2. Enter Load Parameters

    Navigate to the "Load" settings section. You'll find 3-stage programming plus morning-on settings. Each stage controls duration and power percentage.

  3. Map Power % to Current

    The controller regulates current, not voltage. 100% equals full rated current (900mA default, adjustable to 5600mA on DM160). To achieve 1500mA at 35V, calculate the percentage using the Power Calc tab.

  4. Configure All Stages

    Set Stage 1, 2, and 3 to your target percentage. Set Stage 1 duration to 8-12 hours (full night coverage), set Stages 2 & 3 to 0h. This locks current constant throughout the night.

  5. Transmit Settings

    Aim the remote at the controller and press SEND/CONFIRM. The controller will emit an audible beep or LED flash to confirm successful programming.

  6. Enable Intelligent Power

    Activate "Intelligent Power" mode in the remote settings. This automatically reduces LED current if battery voltage drops below ~50% SOC, preventing deep discharge damage—the primary cause of premature battery failure.

Output Current Range

The DM160-U output current is adjustable from 70mA to 5600mA (default 900mA). Your target of 1500mA operates well within specification. Output voltage auto-adjusts to match your LED series—35V is comfortably within the 60V maximum.

Your Configuration Analysis

At 35V × 1500mA = 52.5W LED load: Accounting for 93% efficiency, battery sees ~56.5W draw. A typical 50Ah / 24V battery (~720Wh usable at 50% DoD = 360Wh) provides approximately 6.4 hours runtime. At 3000mA (105W), runtime drops to ~3 hours. Halving the current doubles the operational duration.

Installation Procedure

Critical: Wiring Sequence

Always connect in this exact order: 1. LED Load → 2. Battery → 3. Solar Panel. Disconnect in reverse order. Incorrect sequence may result in permanent controller damage.

24V System Checklist

  1. Verify LED Series Count

    24V systems require minimum 10 LEDs in series (Vout ≥ 30V). Your 35V PCB satisfies this requirement. Confirm all LEDs are wired in series, not parallel branches.

  2. Connect LED Load First

    Connect LED PCB to LOAD terminals (Red = Positive, Black = Negative). Verify polarity with multimeter before applying power.

  3. Connect Battery

    Connect 24V battery to BAT terminals. Battery indicator should display steady illumination. Fast flashing indicates over-discharge condition—charge battery before proceeding.

  4. Connect Solar Panel Last

    Solar panel open-circuit voltage must be ≤ 60V. Connect to PV terminals. During daylight, charging indicator should display slow flash pattern (charging active).

  5. Configure Battery Chemistry

    Use CU-ALL remote to select Lead-acid or Lithium profile. Lithium settings support adjustable charge voltages (7.5–15.5V × 2 for 24V). Default boost voltage: 14.4V × 2 = 28.8V for lead-acid systems.

Power % to Current Mapping

The power percentage setting scales the current output, not direct wattage. 100% equals the maximum configured current (up to 5600mA on DM160). Default factory current is 900mA—you may need to first configure maximum output current to 5600mA, then set 27% to achieve 1500mA. Use CU-ALL to set both parameters.

Status Indicators

Charging Indicator

  • Steady On Solar voltage above threshold (daylight, no charging needed)
  • Off Solar voltage below threshold (night or shaded)
  • Slow Flash Charging in progress—normal operation
  • Fast Flash System over-voltage—check panel Voc

Battery Indicator

  • Steady On Battery connected, normal state
  • Off Battery not connected—check wiring
  • Fast Flash Battery over-discharged—charge immediately

Load Indicator

  • Steady On LED load active and operational
  • Slow Flash Open circuit—check LED connections
  • Fast Flash Short circuit—disconnect load immediately
  • Off Load inactive (daytime or scheduled off)
