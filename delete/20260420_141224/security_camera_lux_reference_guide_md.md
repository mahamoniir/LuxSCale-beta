# 📘 Key Design Rules

## 1️⃣ Vertical Lighting is Critical for Faces
Facial recognition depends primarily on **vertical illuminance**, not horizontal.

Typical face height measurement:

```
1.5 m above ground level
```

---

## 2️⃣ Minimum Recommended Lighting Targets

| Security Level | Recommended Lux |
|----------------|----------------|
| Detection Only | 1 lux vertical |
| Recognition | 2.5 lux vertical |
| Identification | 5 lux vertical |
| High Security | 20–50 lux vertical |

---

## 3️⃣ Color Rendering Index (CRI)

| CRI Level | Performance |
|-----------|-------------|
| 65+ | Minimum usable |
| 80+ | Recommended |
| 90+ | High accuracy |

Higher CRI improves:

- Skin tone accuracy  
- Clothing recognition  
- Vehicle color identification

# ⚠️ Professional Design Reminder

Lighting alone does not guarantee usable video.

Performance also depends on:

- Camera resolution  
- Lens focal length  
- Compression settings  
- Mounting height  
- Viewing angle  
- Environmental conditions  

Always validate lighting using:

```
On-site lux meter measurements
```

---

# 📌 Recommended Measurement Method

Measure lux:

```
At camera target location  
At face height  
Facing camera direction
```

This ensures reliable identification-quality footage.
